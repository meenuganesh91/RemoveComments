
Description of files:

removeComments.h - Header file for removeComments function

removeComments.c - Implementation of removeComments function. The function ignores the newline at the end of a single line comment.

testRemoveComments.c - Calling function multiple number of times for a variety of test cases. The output string is checked against an expected string and outputs whether the test cases succeeded.

testRemoveCommentsWithInput.c - Call removeComments function with input passed as command line input. 

runTest.sh - Compiles and runs testRemoveComments.c. To run './runTest.sh'

runTestWithInput.sh - Compiles and runs testRemoveCommentsWithInput.c.  Multiple inputs are passed within this. Please add any test cases here if necessary. To run './runTestWithInput.sh'
